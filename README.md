# Customer Geospatial Visualization

Aplikasi web satu halaman (Single Page Application) minimalis untuk memvisualisasikan data lokasi koordinat pengunjung atau *customer base* secara interaktif. Proyek ini menangkap data interaksi atau akses pengguna yang masuk ke website, lalu memetakan posisinya secara *real-time* berbasis koordinat geografis.
